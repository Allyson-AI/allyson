## Authentication

You will need to use an API key to authenticate your requests to the Allyson API. You can create an API key in the [Allyson API Console](https://allyson.ai/api-console).

## Usage

You can track all your API sessions and daily cost [here](https://allyson.ai/api-console).

## Costs

Pricing is based on the number of steps during a session they are debited instantly from your balance during the session. You can view the pricing calculator by scrolling to the bottom [on this page](https://allyson.ai/settings).

## Rate Limits

You are limited to **250 requests / minute**, if you need a higher rate limit please reach out to us.
