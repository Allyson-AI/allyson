### Web & Mobile

1. Sign up on [Allyson](https://www.allyson.ai) and then visit the [App Store](https://apps.apple.com/us/app/id6593659141) to download the app.
2. Login in the mobile app & enable the notifications permission.
3. Add funds (minimum $5) to your balance to start spinning up as many sessions as you want.

### API

To begin using Allyson, follow these steps:

1. Sign up on [Allyson](https://www.allyson.ai).
2. Navigate to the API section and create an API key.
3. Once you have obtained your API key, return to the documentation and click on the 'API Reference' to learn how to integrate it into your applications.
