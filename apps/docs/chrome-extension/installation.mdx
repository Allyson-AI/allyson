## Download from Chrome Web Store

To get started with the Allyson Chrome Extension, download it from the [Chrome Web Store](https://chromewebstore.google.com/detail/allyson-ai-web-agent-pay/kcngcogbkiblljbjjfnojmgfppddockf). Simply search for "Allyson Chrome Extension" and click "Add to Chrome" to install it.

## Automatic Authentication

Once installed, the Allyson Chrome Extension automatically authenticates you if you're logged into the [Allyson web app](https://allyson.ai). This seamless integration ensures that you can start using the extension's features without any additional login steps.
