### Session Management
- **View All Sessions**: Easily access and manage all your sessions.
- **Start New Sessions**: Initiate new sessions to handle multiple tasks in parallel.

### File Management
- **Create and Save Files**: Allyson can create and save files, ensuring you have accurate and easily accessible documents.
- **View and Download Files**: View and download your documents directly from the app.

### API Features
- **REST API**: Utilize a robust REST API for seamless integration.
- **API Console**: Manage your API sessions separately from regular sessions.
- **API Keys and Sessions**: Create as many API keys and sessions as needed.
- **Usage Tracking**: Monitor your API usage and spending.
